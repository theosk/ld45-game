■ 変更点 ■

  o 必要システムが65536色(16Bitハイカラー)以上になりました。
    256色の環境でも起動できますが、色化けが起こります。

  o アニメーションウィンドウで、設定のロードとセーブを出来るようにした。
    設定ファイルの拡張子は ias。
    iasファイルのエクスプローラーからアニメーションウィンドウへの
    ドラッグ＆ドロップも出来ます。
    画像ファイルを開く時に同じ場所にiasファイルがあれば、それを開きます。

  o 数値入力ダイアログで最小値が0の時、最小値が表示されないバグを直した。


■ 概要 ■

　o 256色以下のドット絵を描くツールです。
  o 詳しい使い方は idraw3.hlp をご覧ください。
  o フリーソフトです。
  o 転載、再配布を禁止しています。
  o UNIXの idraw とは別物です。


■ 必要システム ■

　ＯＳ  ：日本語 Microsoft Windows95。
　表示色：65536色(16Bitハイカラー)以上。
　ＣＰＵ：ペンティアム90ＭＨｚぐらい。
　メモリ：32メガバイトぐらい。
　解像度：800x600ピクセルを前提に作ってあります。

　ハードディスクに十分な空き容量が必要です。
  Windows98, WindowsNT, Windows2000 での動作確認はしていません。


■ インストール方法 ■

  専用フォルダを作って、そこへ解凍します。


■ ファイル説明 ■

  IDraw3.exe..........本体
  IDraw3.hlp..........ヘルプ(Ver3.32用)
  IDraw3.mem..........文字入力ウィンドウが使う
  Pal16.bmp...........多分、標準16色パレット
  Readme.txt..........このファイル


■ 分かっているバグ ■

  o パレットに同じ色があると、いろいろ不都合がおきます。

■ 履歴 ■

 3.21 o 256色アイコンでファイルに保存できるようにした。
 3.22 o PNG ファイルの読み書きをサポート。
      o ファイルダイアログは表示位置を記憶し、高さを変えられるようにした。
 3.23 o 3.22 で発生する縁描画のバグを直した。
 3.24 o ファイルダイアログで、ファイル名がある状態でファイルの場所ボタンを
        使って移動するとそのファイル名で OK したことになるバグを直した。
      o 拡大・縮小ダイアログをちょっと変えた。
 3.25 o アニメーションウィンドウは1〜4倍で表示できるようにした。
      o 3.21 で遅かった、アイコンとして保存ダイアログの表示の高速化。
      o マス目(大)は幅と高さを別々に設定できるようにした。
 3.26 o シフト移動機能を付けました。
 3.27 o 選択範囲の周りを回っている点線が表示されないバグを直した。
 3.28 o AVIファイルの作成。
      o シフト移動用サブウィドウを付けました。
      o 位置ウィンドウは１〜４倍で表示できるようにした。
      o マウスの右ボタンを離してもコンテキストメニューを閉じないようにする
        オプションを付けました。
      o アニメーション選択中にパレットRGBを変えるとおかしくるバグを直した。
 3.29 o アイコンとして保存する時、今までは色２が透明になりましたが、
        今バージョンから、左下のピクセルの色を透明にするオプションが
        付きました。
        [設定]-"アイコンとして保存する時の透明色"で設定出来ます。
        アイコンとしてクリップボードへコピーする時も、このオプション
        を使います。
      o アイコンとして保存する時に起こるいろいろなバグを直したつもり。
      o ファイル保存ダイアログに「上書き確認」オプションを付けました。
        チェックを外すと上書き確認ダイアログが出なくなります。
      o [編集]メニューにあった、シフト移動関連の項目が無くなりました。
        ショートカットを使ってシフト移動することが出来なくなりました。
      o アニメーションウィンドウの内部を改良したので、それに関係する
        新しいバグがあるかもしれません。
 3.30 o マス目小の表示状態にかかわらず、マス目大を表示できるようにした。
      o 範囲選択ツール使用時に右クリックで出るコンテキストメニューを
        キーボードで表示できるようにした。
        [設定]-[キーボード] のページ"ショートカット"の
        "コンテキストメニューの表示"にキーを割り当てて下さい。
      o アイコンとして保存ダイアログで、アイコンを選んだ時にエラーになる時
        があるバグを直した。
 3.31 o アニメーションウィンドウでAVIで保存する時のバグを直した。
